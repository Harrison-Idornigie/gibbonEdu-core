# GibbonEdu Plugin Development Curriculum

Welcome to the GibbonEdu Plugin Development Course! This curriculum is designed to teach you how to create and maintain plugins (modules) for the GibbonEdu platform.

## Course Structure

### Module 1: Getting Started
- Introduction to GibbonEdu
- Development Environment Setup
- Understanding Module Structure
- The Starter Module Template

### Module 2: Basic Module Development
- Module Manifest and Configuration
- Database Integration (CHANGEDB.php)
- Creating Actions and Pages
- Module CSS and JavaScript

### Module 3: Advanced Development
- Module Hooks and Integration
- QueryableGateways and Domain Logic
- Module Translation
- Security and Permissions

### Module 4: Best Practices
- Code Organization
- Version Control
- Testing
- Documentation

### Module 5: Deployment
- Installation Process
- Updates and Upgrades
- Troubleshooting
- Publishing Your Module

## Prerequisites
- Basic PHP knowledge
- Understanding of HTML/CSS/JavaScript
- Familiarity with MySQL/MariaDB
- Basic Git knowledge

## Getting Started
Each module contains:
1. Lesson content (README.md)
2. Code examples
3. Practical exercises
4. Additional resources

Navigate to each module's directory to begin the lessons in sequence.
